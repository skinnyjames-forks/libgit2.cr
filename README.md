# libgit2.cr

Crystal-lang binding to [libgit2](https://libgit2.org) with inteface similar to [rugged](https://github.com/libgit2/rugged/).

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  git:
    github: smacker/libgit2.cr
```
